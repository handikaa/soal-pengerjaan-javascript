# how to clone the project?

1. copy url "https://github.com/handikaa/soal-pengerjaan-javascript.git"
2. open terminal/cmd on your device go to your folder project or folder to clone the project
3. type "git clone {paste url repository}" and type enter
4. open code edtior and open project folder in your code editor

# How to run the test?

1. Install Node.js if you don't have it installed already.
2. open code editor / IDE
3. open the folder project after cloning on your folder
4. run on your terminal "node {filename}.js" example = "node permutation.js"
